## 章节标签
1. 章节 section
2. 文章 article
3. 段落 p
4. 头部 header
5. 脚步 footer
6. 主要内容 main
7. 旁支内容 aside
8. 划分 div
9. 标题 h1~h6

## 每个标签的全局属性有哪些：
1. class 类属性
2. contenteditable 用户编辑属性
3. hidden 隐藏属性
4. id id选择器属性
5. style css标签
6. tabindex tab按键切换 大1 小0
7. title 鼠标指向提示信息
